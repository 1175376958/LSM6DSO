# ST LSM6DSO 3D accelerometer and 3D gyroscope Sensor Package

ST LSM6DSO 3D accelerometer and 3D gyroscope Sensor I2C extension for makecode.  

Author: shaoziyang  
Date:   2019.Jul  

## Basic usage
```
basic.forever(function () {
    serial.writeNumbers(LSM6DSO.get())
    basic.pause(1000)
})
```
